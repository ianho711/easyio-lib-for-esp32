# 51_WIFI_Scan

## 一、例程简介

此例程摘自IDF的`SCAN`，用来扫描可用的 AP 集（WIFI热点）。

`CONFIG_EXAMPLE_SCAN_LIST_SIZE` 参数需要从示例配置菜单中设置。 它表示驱动程序可以填充的最大扫描列表条目数。

## 三、注意事项

- `Kconfig.projbuild` 中对 `CONFIG_EXAMPLE_SCAN_LIST_SIZE`设定的范围为 0~20，默认值为10。如实际超出，需要手动更改。
